# Dexoria: Pokémon Gallery & Insight Platform

Dexoria is a Pokémon web application that allows users to explore, search, and understand Pokémon through an interactive and visually engaging interface.

Developed as part of the Final Project: The API Integration Bridge, this project demonstrates real-world API integration, modern UI/UX design, and dynamic frontend development using pure web technologies.

##  Features

###  Explore Page
- Animated infinite scrolling Pokémon gallery
- Glassmorphism UI with smooth motion
- Hover interactions and click-to-view details

###  Pokédex
- Search by name or Pokédex number
- Filter by type and generation
- Sort by number or name
- Dynamic Pokémon grid layout
- Load More functionality

###  Pokémon Details
- Modal popup with:
  - Stats
  - Types
  - Abilities
  - Evolutions

###  Insights Hub
- Guides, Top Lists, and Strategy content
- Category filtering system
- Featured insight section

###  Favorites System 
- Save/remove Pokémon instantly
- Stored using localStorage
- Search and sort favorites
- Clear all with confirmation

###  Settings Panel 
- Dark / Light mode toggle
- Accent color customization
- Animation toggle
- Pokédex view preferences
- Search behavior settings
- Persistent user preferences

### UI / UX
- Futuristic gradient design
- Responsive layout (mobile, tablet, desktop)
- Smooth animations and transitions

##  How It Works

1. **API Integration**  
   Pokémon data is fetched dynamically from the PokéAPI using `fetch()`.

2. **Dynamic Rendering**  
   JavaScript builds UI components such as cards, grids, and modals in real time.

3. **Local Storage**  
   User preferences and favorites are stored in `localStorage`, allowing persistence across sessions.

4. **Single Page Application**  
   Navigation between pages (Home, Explore, Pokédex, etc.) is handled without reloading.
