Overview

Dexoria is a fully client-side Pokémon web experience designed to blend data-driven exploration with beautiful, futuristic UI design. Whether you're a long-time fan or new to the world of Pokémon, Dexoria gives you everything you need. From full Pokédex listings to competitive strategy guides.

The goal was simple: build a Pokémon site that actually feels worth exploring. Not just a data table, but a visual journey.

Features

Animated infinite Pokémon gallery with glass-card rows
Full Pokédex with search, type, generation filters
Detailed Pokémon modals. tats, abilities, evolutions
Insights hub: guides, top lists, battle strategy
LocalStorage favorites system with sorting and search
Dark/Light mode with accent color customization
Full settings panel with persistent preferences
Responsive design for desktop, tablet, and mobile

How It Works

Dexoria is entirely client-side. No backend, no database. Here's the simple flow:

  1. Data Fetching: Pokémon data is pulled live from the PokéAPI REST API using fetch().
  2. Dynamic Rendering: JavaScript dynamically builds cards, grids, and modals from API responses.
  3. Local Storage: Favorites and settings are saved in the browser's localStorage, they persist across sessions.
  4. Pure Frontend: No frameworks. Just HTML, CSS, and JavaScript.

Insights Section

The Insights section goes beyond raw data. It provides guides for beginners, ranked top lists for fun exploration, and strategy content for those looking to understand competitive Pokémon. Each card is categorized and filterable so you can jump straight to what interests you.
